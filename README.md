# ⛽ argentina-fuel-analysis

Análisis de Consumo de Combustibles para la Toma de Decisiones Estratégicas de Expansión en Argentina.

## Breve descripción

Se desarrola un prototipo para analizar datos de consumo de combustibles en Argentina.  
Incluye datasets crudos y procesados, notebooks de Colab y un enlace al dashboard interactivo en Google Looker Studio.

## Funcionalidades

- Descarga de archivos CSVs y XLSX de fuentes oficiales actualizados.
- Limpieza y generación de archivos CSVs para análisis de datos.
- Dashboard interactivos para observación del comportamiento de consumo.

## Stack

- Colab
- Google Sheets
- Python
- Looker Studio
